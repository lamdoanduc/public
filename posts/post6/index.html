Chi-square distribution and hypothesis testing using Chi-statistic

April 10, 2021

This post is to describe Chi-square distribution and its applications in goodness-of-fit test and hypotheis testing.

1. What is Chi-square?

  • Assume that we observe an example of a categorical variable that has N different values, 1 through N. Each value has an expected (theoretical) frequency and an observed frequency. We can display them as the following table:
value 1 value 2 … value N
Observed frequency O_1 O_2 … O_N
expected frequency E_1 E_2 … E_N
  • Chi-Square is the sum of the squared difference between the observed and expected frequency divided by the number of values of the variable: $$\chi^2 = \sum^{N}_{j=1} \frac{(O_j - E_j)^2}{E_j}$$

2. What is Chi-square distribution?

  • If Z1, …, Zk are independent, standard normal random variables, then the sum of their squares $$Q=\sum^{N}_{j=1} Z_i^{2}$$ is distributed according to the chi-square distribution with k degrees of freedom.

  • “The chi-square distribution with k degrees of freedom is the distribution of a sum of the squares of k independent standard normal random variables.” - Wikipedia

  • The probability distribution function: $$ f_k(x) = \frac{1}{\sqrt{2^k}\varGamma(k/2)} \frac{\sqrt{x^k}}{x} \frac{1}{\sqrt{e^x}} $$ Where $$\varGamma$$ is a gamma function.

  • Chi-square distribution is widely used probability distributions in inferential statistics, notably in hypothesis testing and in construction of confidence intervals.

import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import chi2
#x-axis ranges from 0 to 20 with .001 steps
x = np.arange(0, 30, 0.002)
#define multiple Chi-square distributions
plt.plot(x, chi2.pdf(x, df=4), label='df = 4')
plt.plot(x, chi2.pdf(x, df=8), label='df = 8')
plt.plot(x, chi2.pdf(x, df=12), label='df = 12')
#add legend to plot
plt.legend()
  • Note: The chi-square distribution approximately converges to the normal distribution as degree of freedom, df, (or k) is large.

3. Goodness-of-fit test

Problem:

  • A report shows that the historical rate of birth in a hospital is different in different seasons:

    Winter Spring Summer Fall
    % of annual birth 15% 25% 30% 30%

  • The total numbers of births by season last year were collected.

Winter Spring Summer Fall
Observed number of births 45 48 55 52
  • We want to see if the data from the last year supports the statement of the historical rate of birth by season? Or in other words, if the historical data is correct?
  • Approach: goodness-of-fit test will be performed to answer the question.

Set up hypothesis

  • H_0: Null hypothesis: the last year data follows the historical rate of birth by season. Or the historical data is correct?
  • H_a: Alternative hypothesis the historical data is NOT correct?
  • From the historical birth rate we can calculate the expected numbers
Winter Spring Summer Fall
Observed number of births 45 48 55 52
Expected number of births 30 50 60 60

Calculate chi-square statistic

  • chi-squared statistic: $$\chi ^2 = \sum^{N}_{j=1} \frac{(O_j - E_j)^2}{E_j} $$

Insert the numbers from the table into the formula, we get: $$\chi ^2 = 9.06$$

Critical value comparison

  • Chose significance level to be 5%, or alpha = 0.05
  • The critical value with 5% of Significance level is 7.815
  • chi-square statistic (9.06) is larger than the critical value (7.815) so we can reject the null hypothesis.

Implement the hypothesis testing in Python

from scipy.stats import chi2_contingency
from scipy.stats import chi2
# observed and expected value table
table = [[45, 48, 55, 52],
         [30,  50, 60, 60]]

# calculate chi-square statistic
chi_square_stat = 0
for i in range(0,len(table[0])):
    chi_square_stat += ((table[1][i] - table[0][i])**2)/table[1][i]
chi_square_stat = round(chi_square_stat,3)

# interpret test-statistic
prob = 0.95
df = 4 - 1
critical = chi2.ppf(prob, df)
print('probability=%.3f, critical=%.3f, stat=%.3f' % (prob, critical, chi_square_stat))
if abs(chi_square_stat) >= critical:
    print('Reject H_0')
else:
    print('Fail to reject H_0')

4. Chi-Square Test

Problem:

  • We can use Chi-Squared Testing to see if two categorical variables are independent. Go back to the example in section 3, the data of births in the hospital last year, we want to see if the gender of the newborn baby and season are independent or not. The contingency table shows the number of different genders of the babies by the season:
Winter Spring Summer Fall
Male 15 20 27 20
Female 30 28 28 32

Approach:

We are going to use Chi-Squared Test implemented in Python.

  • H_0: Null hypothesis, the two variables are independent. H_a: Alternative hypothesis, the two variables are dependent.
  • Expected values and chi-square statistic are calculated based on the contingency table using the chi2_contingency function.
  • We chose significance level to be 5%.
from scipy.stats import chi2_contingency
from scipy.stats import chi2
# observed and expected value table
con_table = [[15, 20, 27, 20],
         [30,  28, 28, 32]]
print(table)
statistic, p_value, df, expected = chi2_contingency(con_table)
print(expected)
# interpret test-statistic
probility = 0.95
critical_value = chi2.ppf(probility, df)
print('probability=%.2f, critical=%.2f, stat=%.2f' % (probility, critical_value, statistic))
if abs(statistic) >= critical_value:
    print('Dependent (reject H0)')
else:
    print('Independent (fail to reject H0)')
